W-SAGE

Web tool for Spatial Analysis of GEodata

Sobre o W-SAGE

Esta é uma ferramenta para análise de dados desenvolvida por Raul Sena Ferreira como parte de seu trabalho de conclusão de curso. Ela foi inicialmente concebida para analisar dados geográficos mas também pode ser usada para analisar dados discretamente. Possui um painel administrativo com funcionalidades como:

Busca baseada em localização geográfica;
Busca baseada em características específicas dos dados;
Busca baseada em polígonos;
Permite salvar as consultas frequentes;
Visualização de dados em mapa de calor (Heatmap);
Visualização de dados baseada no CEP (ponto);
Visualização de dados em gráficos (Pizza, Histograma e etc);
Visualização de Logs, dentre outros.

A ferramenta permite que o usuário altere suas configurações de acesso ao banco de dados fazendo com que seja possível a integração com bases de dados diferentes.
Embora a ferramenta esteja pronta pra uso deve-se levar em consideração que esta versão é experimental e ainda está sob desenvolvimento e seu progresso pode ser acompanhado através do GitHub do desenvolvedor.
